## 🧠 Key Concepts Learned So Far

### Linux Fundamentals ✅
- Navigation: `ls`, `cd`, `pwd`, `find`, `locate`
- File permissions: `chmod`, `chown`, `rwx` notation
- Process management: `ps`, `top`, `kill`
- Package management: `apt`, `apt-get`
- Text processing: `cat`, `grep`, `awk`, `sed`

### Networking (In Progress)
- OSI Model — 7 layers and their functions
- TCP vs UDP — when and why
- Common ports: 22 (SSH), 80 (HTTP), 443 (HTTPS), 3389 (RDP)
- DNS resolution process
