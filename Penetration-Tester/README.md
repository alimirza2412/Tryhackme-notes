## 🔍 What is Penetration Testing?

Penetration testing (pentesting) is the practice of **legally and ethically attacking systems** to find vulnerabilities before malicious hackers do. A penetration tester follows a defined methodology:

```
1. Reconnaissance    →  Gather info about the target
2. Scanning          →  Identify open ports & services
3. Exploitation      →  Attack the vulnerabilities found
4. Post-Exploitation →  Maintain access, escalate privileges
5. Reporting         →  Document findings professionally
```

## 🧰 Core Pentesting Tools

| Tool | Category | Purpose |
|------|----------|---------|
| 🔍 Nmap | Reconnaissance | Port scanning, OS detection, service enumeration |
| 🕷️ Burp Suite | Web Testing | Intercept & manipulate HTTP traffic |
| 💣 Metasploit | Exploitation | Run exploits, manage sessions, post-exploitation |
| 📁 Gobuster | Web Recon | Brute-force directories, DNS subdomains |
| 🐍 SQLMap | Web Testing | Automated SQL injection detection & exploitation |
| 🔑 Hydra | Brute Force | Password cracking against various protocols |
| 🦅 Nikto | Web Scanning | Web server vulnerability scanner |
| 🐧 Kali Linux | OS | Purpose-built pentesting Linux distribution |
